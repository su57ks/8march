> Интерактивная страница-поздравление с 8 марта. Кликай на цветок, собирай букет, открывай казино и наслаждайся анимацией!

![Preview](preview.png)

## ✨ Возможности

- 🎨 **7 уникальных тем оформления** — от нежной розовой до космической
- 🌸 **Кликер-игра** — собирай цветы, нажимая на большой цветок
- 🎰 **Казино** — умножай свои цветы (открывается при 1000 цветов)
- 🌼 **Гадание на ромашке** — отрывай лепестки и узнавай судьбу
- 🎁 **Промокоды** — секретные коды с приятными бонусами
- 💐 **Цветопад** — красивая анимация падающих цветов
- 🌟 **Секретные темы** — разблокируй золотую, космическую и темно-синюю

## 🎮 Как играть

1. **Кликай на большой цветок** — собирай цветы
2. **Каждые 100 цветов** — праздничная анимация
3. **Собери 1000 цветов** — открой казино и золотую тему
4. **Играй в мини-игры** — Ромашка (10🌸) и Букет (10🌸)
5. **Ищи промокоды** — нажми 5 раз на подпись автора

## 🎨 Темы оформления

| Тема | Цветовая схема | Как открыть |
|------|---------------|-------------|
| Розовая | 🌸 Нежные розовые тона | По умолчанию |
| Белая | ⚪ Светлая, минималистичная | В настройках |
| Сиреневая | 💜 Лавандовые оттенки | В настройках |
| Ярко-розовая | 💗 Насыщенный розовый | В настройках |
| Золотая | 👑 Благородное золото | 1000🌸 или промокод |
| Космическая | 🌌 Глубокий фиолетовый | Промокод 505 |
| Темно-синяя | 🌃 Ночное небо | Промокод 1337 |

## 🎁 Промокоды

| Код | Эффект |
|-----|--------|
| `52` | +5200 цветов |
| `67` | +1 цветок |
| `69` | +690 цветов |
| `505` | +505812 цветов + космическая тема |
| `1337` | +133700 цветов + темно-синяя тема |
| `1488` | 🚫 Обнуление счета |
| `МЫВАСЛЮБИМ` | 💖 +1 000 000 000 цветов |
